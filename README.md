# Saurabh Saini - Senior MERN Stack Developer Portfolio

This repository contains my personal portfolio website, featuring a production-ready, high-performance web application designed to highlight my core engineering capabilities, scalable architectures, and 4.5+ years of industry experience.

The application is completely containerized using **Docker** and configuration structures are documented for localized orchestration testing.

## 🛠️ Technology Stack

- **Frontend:** React.js, Next.js, Redux (Toolkit/Saga), RTK Query, HTML5, CSS3, Tailwind CSS
- **Core Backend Architecture Knowledge:** Node.js, Express.js, REST APIs, Microservices, Cron Jobs
- **Databases & Caching Infrastructure:** MongoDB (Mongoose), MySQL (Connection Pooling), Elasticsearch, Redis, BullMQ
- **DevOps, Security & Deployment:** Docker, Docker Compose, PM2 (Cluster Mode), JWT, RBAC, Zod, Helmet, Vercel

## 🐋 Getting Started (Docker Setup)

To run the entire portfolio layout locally in an isolated container environment:

### Prerequisites

Make sure you have **Docker Desktop** installed and actively running on your machine.

### Launching the Application

1. Clone the repository and navigate to the root directory:
   ```bash
   git clone [https://github.com/Saurabh-Saini/portfolio.git](https://github.com/Saurabh-Saini/portfolio.git)
   cd portfolio
   ```
